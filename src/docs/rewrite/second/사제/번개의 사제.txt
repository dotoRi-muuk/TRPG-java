번개의 사제

판정 사용 스탯 : 지능(지혜)

기본 공격(기본 공격) : 대상에게 1D6의 데미지를 입힙니다.

축복 (패시브) : 자신이 공격 성공 시 이번 턴 아군의 데미지가 1.5배로 증가합니다.
신성한 육체 (패시브) : 자신의 지능, 지혜 스탯이 +3 증가합니다.
신뢰 (패시브) : 아군 1명의 모든 스탯이 +2 증가합니다.
믿음 (패시브) : 아군 판정 실패 시 본인이 다음 턴 행동 불가를 얻는 대신 지능 판정을 진행합니다. 성공 시 아군에게 재판정 기회를 지급합니다.

스파크 (스킬) : 대상에게 2D4의 피해를 입힙니다. (마나 1 소모)
체인 라이트닝 (스킬) : 최대 3명의 대상을 지정합니다. 적에게는 2D6의 피해를 입히고, 아군에게는 2D4의 보호막을 부여합니다. (마나 3 소모, 쿨타임 3턴)
일렉트릭 필드 (스킬) : (최대 6턴 영창) (광역) 아군의 데미지를 50n% 증가시키고, 적에게 (8-4n)D4의 피해를 입힙니다. (마나 8 소모, 쿨타임 7턴)
스트라이크 (스킬) : 대상에게 2D8의 피해를 입힙니다. (마나 2 소모, 쿨타임 5턴)
일레이스터 (스킬) : 아군 1명의 데미지를 다음 턴까지 2배로 증가시킵니다. (마나 5 소모, 쿨타임 5턴)
신뇌격 (스킬) : 대상에게 3D20의 피해를 입힙니다. (영창 2턴, 마나 7 소모)

독점 (스킬) : 아군 데미지 n% 증가 효과를 본인 데미지 n+100% 증가 효과로 변경합니다. 전투 시작 시 자신 외의 아군이 1명 이상 존재할 때 사용 가능합니다. 전투 내 영구 지속되며, 취소할 수 없습니다. (마나 7 소모)
군림 (스킬) : [신성한 육체] 효과를 제거합니다. 아군의 힘, 민첩, 지능을 3 감소시키고, 자신의 힘, 민첩, 지능을 5 증가시킵니다. 전투 시작 시 자신 외의 아군이 1명 이상 존재할 때 사용 가능합니다. 전투 내 영구 지속되며, 취소할 수 없습니다. (마나 9 소모)
불신 (스킬) : [신뢰] 효과를 제거합니다. (전투 시작 시점의 아군 수) - (현재 아군 수) 만큼 아군의 스탯을 감소시키고, 본인의 스탯을 증가시킵니다. 전투 시작 시 자신 외의 아군이 1명 이상 존재할 때 사용 가능합니다. 전투 내 영구 지속되며, 취소할 수 없습니다. (마나 8 소모)
배신 (스킬) : [믿음] 효과를 제거합니다. 지능, 지혜 판정 실패 시 자신 외 모든 아군이 다음 턴 행동 불가 상태를 얻는 대신 재판정 기회를 얻습니다. 전투 시작 시 자신 외의 아군이 1명 이상 존재할 때 사용 가능합니다. 전투 내 영구 지속되며, 취소할 수 없습니다. (마나 13 소모)
신앙심 (스킬) : 최대 3턴 동안 자신과 적 1명에게 행동 불가를 부여합니다. 지속 시간 중 모든 아군의 데미지가 1.5배로 증가합니다. (마나 6 소모, 쿨타임 10턴)
