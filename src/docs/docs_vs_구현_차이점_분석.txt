이 문서는 src/docs 폴더 내의 직업 정보와 src/main/primary, src/main/secondary 폴더 내의 
Java 구현 간의 차이점을 스킬/기술 하나하나 상세히 분석한 결과입니다.

작성일: 2024년

================================================================================

### 1.1 전사 (Warrior.java)

#### 구현된 항목:
✅ 기본공격 (D6)
✅ 광란 패시브 - (시작 체력 - 현재 체력) / 5 만큼 추가 데미지
✅ 강타 (D10, 스태미나 2)
✅ 가로베기 (D8, 최대 3명 타겟)
✅ 육참골단 전용 수비 - 받는 피해 150%, 반격 200%

#### 미구현/차이점:
❌ 도발 스킬 미구현
   - docs: 2턴 동안 적 도발, 상대에게 1턴 수비 불가, 받는 데미지 2 감소
   - 마나 3, 쿨타임 15턴
❌ 가로베기의 "최대 3명 타겟" 로직은 코드에서 데미지 출력만 하고 
   실제 타겟 처리는 외부에서 해야 함


### 1.2 궁수 (Archer.java)

#### 구현된 항목:
✅ 기본공격 - 명사수 패시브 적용
   - 단일 스탯 사용 시 D4
   - 힘과 민첩 동시 사용 시 D6
✅ 퀵샷 (3xD4, 주사위당 -1, 스태미나 5)

✅ 사냥감 스킬 - 연속 공격 배율 (1회:100%, 2-3회:125%, 4-6회:150%, 7회+:200%)

#### 미구현/차이점:
❌ 기회 스킬 미구현
   - docs: 원하는 순간을 자신의 턴으로 만들 수 있다

   - 마나 7, 쿨타임 25턴
⚠️ 명사수 패시브의 "수비효과 발동 X" 로직은 게임 시스템에서 처리해야 함
