빛 -> 여명 광휘 정화 석양
어둠 -> 침잠 흡수 공포 흑염
시간 -> 기억 순환 정체 찰나
영혼 -> 영혼묘 사념 자아 잔향
번개 -> 천둥 격동 섬광 폭주

여명 -> 여명의 성녀 (리에) - 다음 턴까지 하나의 스탯을 0으로 감소, 감소한 값 만큼 다른 스탯 증가
광휘 -> 자비의 대사제 - 이번 턴 모든 아군 받는 데미지 50%
생명 -> 세계수의 폭주 (아일리시아) - 자신 또는 아군 1명의 잃은 체력의 30% 회복
석양 -> 대리자의 추억 (헤이타) - 다음 턴까지 받는 데미지 150% / 가하는 데미지 250%

공간 -> 용사의 소문 (사념 균열) - 자신을 대상으로 하는 공격을 회피 후 턴 획득
초월 -> 마왕의 부활 (아카르나엘) - 다음 턴까지 받는 데미지 50% / 가하는 데미지 200%
공포 -> 잊혀진 대리자 -> 다음 턴 모든 적 공격 불가 / 받는 데미지 150% 부여
흑염 -> 어둠의 소문 (에트라함) -> D6% 데미지

순환 -> 거슬러진 운명 - 모든 스킬 쿨타임, 순환 초기화
정체 -> 멈춘 시간 속의 추억 - 이번턴 모든 적 행동불가
찰나 -> 불가항력 - 이번 턴 행동 2회
기억 -> 이별의 하루 (에트) - (현재 턴) ~ (현재 - 5)턴 중의 원하는 턴으로 자신의 상태를 되돌린다

사념 -> 용사의 소문 (사념 균열) - 이번 턴 적 1명 행동불가 부여 / 해당 적의 공격 기술/스킬을 1개 골라 사용할 수 있음 (일부 불가)
자아 -> 잃어버린 나 - D4 < (4 - 지속 턴)이라면 행동불가 부여 / 실패한 턴 자신에게 행동불가 부여
잔향 -> 흩어져가는 길 - 지난 3턴 동안 적이 입은 피해의 50%만큼 피해 부여
추억 -> 죽음 너머의 인연 - 이전에 처치한 적의 영혼을  2턴후까지 소환 (소환한 영혼은 자신 다음으로 행동하며 영혼의 행동으로 감소하는 스태미나, 마나는 소환자가 감당한다.)

천둥 -> 천둥에 다다랐던 전사 (호아킨) - 이번 턴 적 공격 불가 / 가하는 데미지 200% (턴 소모X)
격동 -> 멈추지 않는 투지 - 최대 5턴 동안 모든 적의 우선 공격 타겟을 자신으로 설정 / (효과 지속 턴50%) 데미지 추가 / 사용 후 처음으로 맞는 공격의 피해 300% / 피격 시 효과 해제
섬광 -> 미련 없는 전우 - 다음 턴까지 모든 적 받는 데미지 200% / 수비 불가 부여
폭주 -> 사라진 이성 속의 인연 - 최대 6턴 동안 지속 / 전체 턴 시작 시 D6 < (6 - 지속 턴) 성공 시  이번 턴 데미지 250% / (지속 턴) < 4 일 때 실패 시 다음 턴까지 행동불가

빛 -> 대리자의 추억(빛) - 전투 당 1회 아군 1명의 잃은 체력 모두 회복 / 이번 턴 1개의 스탯 100% 증가, 데미지 200%, 적 행동불가,
어둠 -> 대리자의 추억(어둠) - 전체 턴 시작 시 사용 가능 / 자신 제외 모든 아군 D20% 데미지(보정X) / 1명의 대상의 턴을 자신의 턴으로 변경 / 이번 턴 데미지 200%
시간 -> 대리자의 추억(시간) - 모든 스킬 쿨타임, 순환 초기화 / 자신 외에 모두 행동불가 / 행동불가를 부여한 대상만큼 턴 획득
영혼 -> 대리자의 추억(영혼) - 5턴후까지 입힌 데미지의 50% 추가 데미지 / 2턴 후까지 적의 영혼을 소환, 그 시간동안 적은 행동불가 부여 (소환한 영혼은 자신 다음으로 행동하며 영혼의 행동으로 감소하는 스태미나, 마나는 소환자가 감당한다)
번개 -> 대리자의 추억(번개) - 최대 8턴 지속 / 모든 적의 우선 타겟이 됨 / D8 < (8 - 지속 턴) 성공 시 이번 턴 데미지 (효과 지속 턴50)% 데미지 증가 / 공격 3회 피격 시 해제 / (지속 턴) < 5일 때 해제 시 이번 턴 행동불가
